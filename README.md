# FAS-RS 智能调度模块

## 新增特性
- **Zygisk后端支持**：现在兼容4.12-5.10内核设备
- **架构扩展**：支持arm64-v8a/x86_64架构
- **动态后端切换**：根据内核版本自动选择最优方案

## 安装要求
1. 内核版本：4.12及以上
2. 必须使用最新版Magisk（需Zygisk支持）
3. 推荐在工程模式下开启`Zygisk强制挂载`

## 注意事项
- 5.10+内核默认使用eBPF后端
- 4.12-5.10内核自动切换Zygisk模式
- x86设备需确认Magisk已部署x86_64的libzygisk.so

如遇问题请：
1. 检查日志文件：`/sdcard/Android/fas-rs/fas_log.txt`
2. 确保已授予模块所有危险权限
3. 升级Magisk至最新版本
